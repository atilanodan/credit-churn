# credit-churn
# Segmentación de clientes bancarios

Proyecto de análisis y segmentación de clientes usando el dataset **Bank Churners**
(10,127 clientes). El objetivo es identificar perfiles de clientes para entender
mejor el comportamiento de quienes abandonan el banco.

# Herramientas

Python, Pandas, NumPy, Matplotlib, Seaborn, Google Colab
